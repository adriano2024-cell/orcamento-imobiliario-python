## 📌 Sobre o Projeto

Este projeto consiste em um sistema desenvolvido em **Python** para cálculo de aluguel e geração de **orçamento imobiliário em CSV**, aplicando conceitos fundamentais de programação e simulando uma situação real do mercado.

O sistema foi estruturado com foco em **boas práticas, organização de código e lógica de negócio**, tornando-se aplicável tanto em ambiente acadêmico quanto profissional.

---

## 🎯 Objetivo

Automatizar o processo de cálculo de:

- Valor do aluguel
- Taxas administrativas
- Parcelamento de contratos
- Geração de planilhas em CSV

---

## ⚙️ Funcionalidades

✔ Entrada de dados via terminal  
✔ Cálculo automático de valores  
✔ Aplicação de regras de negócio  
✔ Geração de parcelas  
✔ Exportação para arquivo CSV  
✔ Validação de entradas do usuário  

---

## 🧠 Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Classes e métodos
- Métodos abstratos (ABC)
- Estruturas condicionais (`if/else`)
- Laços de repetição (`while`)
- Funções auxiliares
- Manipulação de arquivos CSV
